Why Python


易於學習，語法簡潔

Python 注重的是如何解決問題，而不是複雜的程式語法及結構，使你能夠專注於解決問題，而非糾結於程式語法，非常適合中、小學生學習。

應用領域廣泛，資料科學首選

Python 可以做很多事情。例如 Web 開發、遊戲、自動化維運、資料分析、科學計算、人工智慧、機器學習等等，在資料科學當中更是首選。

重要的程式語言

Python連續幾年都被列為前五大程式語言之一，2018年IEEE Spectrum發布的程式 語言排名更是排名第一。

跨領域學習

許多不同領域的大學科系，都以Python作為學生學習程式語言首選，更有許多先進國家在中、小學推動。
