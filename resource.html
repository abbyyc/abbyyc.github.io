自學資源

自學網站

Coding 是未來最令人嚮往的技能之一。
我們整理出一些國外熱門的程式教學網站，這些網站易於使用，利用遊戲使 Coding 變得有趣和富有想像力。Coding 不僅僅涉及技術技能，更能引導孩子們建立邏輯和創造性思維。透過使用謎題，謎語，遊戲和挑戰，可以培養孩子們解決問題，耐心，和創造力的能力。

code.org

(Recommended for Ages 4 to 15)

這是一個非營利性的基金會網站，也是初學者學習程式一個很好的起點，它共享大量有用的線上資源。

Scratch

(Recommended for Ages 6 to 12)

由麻省理工學院媒體實驗室，針對8至16歲的兒童設計，目前也是許多小學積極推動的課程之一。Scratch易於讓孩子們構建他們夢寐以求的許多東西。 這裡沒有任何模糊的程式碼，將Scratch方塊安排並拼接在一起，就好像它們是虛擬樂高積木一樣，目前已推動多年，易學易懂，國外許多學校都將其放在K12中學習，同時它也是一個小朋友充滿活力的線上社群，大家可以彼此交換想法及作品。

Khan Academy

(Recommended for Ages 14+)

Khan Academy以其廣泛且具有挑戰性的數學遊戲而聞名，它還提供基本的編程教程，教孩子們如何構建圖形、動畫、 互動式及可視化等等。

codecademy

(Recommended for Ages 14+)

Codecademy是一個線上交互式網站平台，它提供免費寫程式課程，利用簡單的練習教孩子基本代碼，感覺像遊戲。

code combat

(Recommended for Ages 10+)

Code Combat適合年齡較大的孩子，使用互動且競爭的遊戲模式來刺激學習。如果想要一邊玩又一邊學習基本的觀念， 可以參考看看。

Tynker

(Recommended for Ages 10+)

該網站允許孩子們按照自己的節奏學習編程，透過簡單的課程，孩子們幾乎可以做任何他們想要的事情，包括應用程序 、遊戲或網站頁面。 該網站是免費使用的，但如果您的孩子希望將他們的專業知識提升到新的水準，則需訂閱。

Blockly

(Recommended for Ages 8+)

顧名思義，這個網站教授一系列謎題和區塊的編程。 孩子們需要通過拖放來解決難題，以創建一個解決特定問題的故事 ， 之後，孩子們將可以看到他們的工作成果。

CODEMOJI

(Recommended for Ages 6-14)

codemoji.com是一個適合6-14歲的兒童青少 年趣味學習平 臺，孩子可以使用eomji學習程式設計，用emoji表情符 號來表示不 同HTML/CSS/JavaScript程式碼塊，來建立網站、動畫或遊戲等等。codemoji想讓孩子學習建立直觀的 程式， 避免繁瑣的語法，幫助 兒學習者強化對概念的理解。

Code Wars

(Recommended for Ages 14+)

該網站專為專為較高年紀的青少年設計，提供一個平台，讓學生通過線上挑選真正的挑戰者，可以進行編碼競賽，有多 種程式語言可供選擇。

Code Avengers

(Recommended for Ages 5-15+)

雖然Code Avengers沒有引人注目的圖形，但它確實提供 了一系列免費的介紹性課程，用於構建網頁，應用程序和 遊戲，根據身分可以至少有7天的免費試用。

Thimble

(Recommended for Ages 5-15+)

Mozilla這個網站允許孩子們透過此平台來學習互動式編程，孩子們可以直接編碼並觀察編碼輸出的內容，它也允許建 立線上網頁。

Code Monster

(Recommended for Ages 5-15+)

Code Monster特別適合兒童學習，利用相鄰兩個區塊， 一個顯示程式碼，另一個顯示結果。 當孩子使用程式碼時( 在每一個提示的幫助下)，將 了解每個命令的作用。
