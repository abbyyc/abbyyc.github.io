關於我們

幫助孩子成功的未來做好準備

未來，Coding 就像數學或閱讀一樣，將成為一種基本技能，我們教孩子們能夠用當今熱門的程式語言 Python，創造出屬於自己的遊戲、應用程式和其它無限的可能性。

在 Coding Lab，我們利用Python結合數學概念，建立邏輯性及系統性的思考，引導孩子們一步一步解決問題。從基本的數學運算開始，隨著不同年級及課程，讓有興趣的孩子們逐步轉向需要更高階數學知識的主題，作為大數據及人工智慧的基石。藉由coding來解決數學問題及生活應用，帶領大家了解計算機科學的抽象及有趣世界。

學習過 coding 的孩子們，將有助於發展他們的創造力、邏輯思維以及解決問題的能力，也能從不同角度去看一件事情，理性地分析問題和資訊，擁有判斷是非對錯的能力。我們陪同孩子們一同探索coding的奧妙，給他們工具，教導他們使用，並學習如何尋找協助，培養終身自學的能力，並運用孩子們的創意思考，創造出屬於他們的未來，Coding Lab 將引領孩子們五力全開。

  • 邏輯運算思維
  • 解決問題能力
  • 數據分析能力
  • 創新應用能力
  • 終身自學能力

Ｗhy python?

為何現在的孩子需要學習電腦編程(Computer Programming)呢？ 因為編程可以促進創造力、邏輯推理及解決問題，幫助孩子們從無到有創造東西，遇到問題也可以學習如何運用邏輯性思考來解決問題。

為何選擇Python呢？幾個重要原因

  • Python易於學習、功能強大且被廣泛使用的高階程式語言
  • Python連續幾年都被列為重要的五大編程語言之一，2018年IEEE Spectrum發布的程式語言排名更是排名第一
  • Python是當今各國大學中最常用的第一語言
  • Python在科學計算、AI、 Web 編程及許多方面處於領先地位

學習Python將使您的孩子能夠構建當今在許多領域廣泛使用的程序; 例如Web開發，數據科學和人工智慧，Python中的命令和語法（必須如何佈局代碼的規則）相對簡單，即使沒有任何編碼經驗，Python也能讓孩子們非常容易上手。 Coding Lab會利用程式與數學的結合，製作適合與青少年學習相關的教材，設計出淺顯易懂的內容，以及循序漸進的實作方式，讓他們從熟悉的領域來學習程式，更能輕鬆地進入Python程式設計的世界，並與世界無縫接軌。
